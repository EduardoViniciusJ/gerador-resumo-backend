Ajuda

Como usar o Gerador de Resumo

  1. Cole ou digite o texto que deseja resumir na caixa de texto.
  2. Clique no botão "Gerar Resumo".
  3. Aguarde alguns segundos enquanto o resumo é gerado.
  4. Visualize o resumo gerado na seção "Resumos Gerados".

Funcionalidades

  • Geração rápida de resumos usando inteligência artificial.
  • Copiar o resumo.

Dúvidas Frequentes

Qual o tamanho máximo do texto?

O limite depende da API utilizada.

Os resumos são salvos?

Os resumos são exibidos na página atual, mas não são salvos permanentemente. Ao recarregar a página, eles serão perdidos.
